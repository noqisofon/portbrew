Description
----------------------------------------------------------------------

RubyPorts は RubyGems にリスペクトされたパッケージ管理ツールであり、パッケージ管理フレームワークです。  
その名の通り Ports の RubyGems 風のラッパーです。

RubyPorts 自体をアップグレードする場合はこの様に打ってください:

    $ sudo port update --system
